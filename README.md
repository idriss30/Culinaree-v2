# Culinaree

Culinaree is a recipe website made with html css and javascript.
It is compiled with webpack and converted with babel.

# Available commands once cloned

npm run start will launch a localhost server. </br>
npm run build will compile the project in order to create a final build.


# Project features
  asynchronous javascript </br>
  Dom manipulation </br>
  Api implementation; </br>
